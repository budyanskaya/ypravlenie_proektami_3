# Управление проектами
3. Оценка и распределение ресурсов

*Ситуация:* У вас есть проект с 5 задачами, требующими разных компетенций (аналитик, разработчик, тестировщик, дизайнер). В распоряжении — 3 сотрудника с частичной загрузкой (50–75%). Бюджет ограничен.

*Задание:* Предложите метод оценки потребности в ресурсах и инструмент для их оптимального распределения. Как вы будете контролировать перегрузку команды и минимизировать простои?
